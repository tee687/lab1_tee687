# Lab 1: Grade Evaluator & Archiver

## Files Included
- `grade-evaluator.py`: Python script for grade calculation.
- `organizer.sh`: Bash script for file management and archiving.
- `grades.csv`: Data source file.

## How to Run
1. **Python Application:**
   Run `python3 grade-evaluator.py` to see the final grade and GPA.
2. **Shell Script:**
   Run `bash organizer.sh` to archive the current grades and reset the workspace.
